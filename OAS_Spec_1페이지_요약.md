## 🏢 CJ 사례 (실사용자 증언)

**윰시 (CJ 개발자):**
> "나 써보니까 진짜 개편해서, 원래는 서버한테 문서 만들어달라고 하잖아.  
> 근데 레포 권한 있으면 걍 프엔도 플젝 읽고 문서 만들어서 바로바로 확인할 수 있어서 좋더라궁"

**CJ의 방식:**
1. 백엔드 코드에서 자동으로 operation.yaml 생성 (CJ 내부 명명, 표준은 openapi.yaml)
2. 프론트가 yaml 기반으로 타입/문서 자동 생성
3. 실시간 API 스펙 동기화
