## 📍 Slide 2: 해결책 - OAS Spec

### OpenAPI Specification이란?
**API의 설계도이자 계약서**

```yaml
# openapi.yaml
paths:
  /news:
    get:
      summary: 뉴스 조회
      responses:
        "200":
          schema:
            type: object
```

**한 번 작성하면:**
- ✅ 타입 자동 생성
- ✅ 문서 자동 업데이트
- ✅ Mock 서버 자동 생성
